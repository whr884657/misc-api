# 一言API接口文档 (v1)

## 接口概述

一言API是一个本地随机一言接口，从本地数据文件中随机获取一条优美的句子。支持按分类获取或随机获取，适用于网站展示、签名档、博客装饰等场景。

## 接口地址

```
GET/POST https://api.xunjinlu.fun/api/yiyan/v1.php
```

## 请求方式

- `GET` - 通过URL参数传递
- `POST` - 通过表单或JSON格式传递

## 请求参数

### 参数说明

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| c | int | 否 | 分类序号（1-12），不传则随机返回 |

### JSON 请求参数示例

```json
[
    {
        "name": "c",
        "type": "integer",
        "required": false,
        "description": "分类序号：1动画 2漫画 3游戏 4文学 5原创 6来自网络 7其他 8影视 9诗词 10网易云 11哲学 12抖机灵，不传则随机返回",
        "example": 4
    }
]
```

#### 请求参数详细说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `c` | integer | 否 | 分类序号，范围为 1-12，不传则随机返回 |

**分类映射表：**

| 序号 | 分类 | 序号 | 分类 |
|------|------|------|------|
| `1` | 动画 | `7` | 其他 |
| `2` | 漫画 | `8` | 影视 |
| `3` | 游戏 | `9` | 诗词 |
| `4` | 文学 | `10` | 网易云 |
| `5` | 原创 | `11` | 哲学 |
| `6` | 来自网络 | `12` | 抖机灵 |

#### 请求示例

**随机获取一言：**
```json
{}
```

**指定分类获取一言（文学类）：**
```json
{
    "c": 4
}
```

## 响应字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| code | int | 状态码，200为成功，400为参数错误，500为服务器错误 |
| msg | string | 状态描述信息 |
| data | object/null | 返回的一言数据 |
| data.id | string | 一言的唯一标识ID |
| data.content | string | 一言内容 |
| data.source | string | 一言的来源/作者 |
| data.category_id | int | 分类序号（1-12） |
| data.category_name | string | 分类名称 |
| api_info | object | API开发者信息 |
| api_info.developer | string | 开发者名称 |
| api_info.blog | string | 开发者博客地址 |
| api_info.api_platform | string | API平台地址 |


## 注意事项

1. **跨域支持**：接口已配置CORS，支持跨域访问
2. **数据随机**：每次请求都会从对应分类中随机返回一条数据
3. **本地数据**：所有数据均来自本地文件，无需依赖外部API
4. **分类统计**：当传入无效分类时，接口会返回所有可用分类列表
5. **编码格式**：接口返回UTF-8编码的JSON数据
