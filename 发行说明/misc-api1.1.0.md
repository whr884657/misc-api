# misc-api 1.1.0 发行说明

**发布日期：** 2026-07-10  
**版本标签：** v1.1.0

## 本版摘要

新增完整用户体系：用户登录、邮箱验证码注册、忘记密码与用户中心；管理员不再开放注册。

## 变更内容

- 新增用户模块（`user/`）：登录、注册、忘记密码、用户中心
- 新增 `vs_user` 数据表（用户名、密码、邮箱、注册时间、最后登录时间）
- 移除 `admin/register.php`，管理员账号仅在安装向导创建
- 用户注册依赖管理员配置的邮箱发信；未配置时页面提示联系管理员
- 安全机制与管理员一致：CSRF、同源校验、登录/邮件频率限制
- 前台首页增加「用户中心」入口

## 数据库变更

- 新增表：`{prefix}user`
- 自 v1.0.0 升级：在线更新或手动执行 `install/migrations/1.1.0.sql`

## 下载

- 发行页：https://gitee.com/xunjinlu/misc-api/releases/tag/v1.1.0
- 更新包：`misc-api1.1.0.zip`

## 升级说明

- 从 v1.0.0 升级：执行在线更新即可自动迁移数据库
- 新安装：访问 `/install/` 完成向导，用户表随安装创建
