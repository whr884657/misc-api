# misc-api 1.4.0 发行说明

**发布日期：** 2026-07-11  
**类型：** 大版本（用户管理功能增强）  
**数据库变更：** 无（封禁复用已有 `user.status` 字段：1=正常，0=封禁）

---

## 本版摘要

- 管理员用户管理新增**封禁 / 解封 / 删除**操作
- 移动端用户卡片大幅紧凑化，OAuth 绑定状态右上角展示
- 电脑端表格新增操作列，保留注册时间与最后登录

---

## 变更详情

### 用户管理（管理员后台）

1. **封禁**：将用户 `status` 设为 0，禁止登录（含 OAuth 与密码登录）
2. **解封**：恢复 `status` 为 1
3. **删除**：永久删除用户记录（含 OAuth 绑定信息）
4. 删除操作需二次确认

### 移动端 UI

- 一行显示：`ID {id} {用户名}`，邮箱次行
- 右上角：未绑定任何第三方时显示「未绑定」；已绑定则显示 QQ/Gitee 图标（1～2 个）
- 隐藏注册时间、最后登录
- 底部紧凑封禁/删除按钮

### 电脑端 UI

- 用户列：`ID xxx 用户名`，封禁用户显示「已封禁」标签
- 保留邮箱、第三方绑定、注册时间、最后登录列
- 末列操作：封禁（或解封）+ 删除

### 登录体验

- 封禁账号尝试登录时提示「账号已被封禁，请联系管理员」

## 升级说明

### 自 1.3.x 在线升级

1. 后台「系统升级」检测并安装 v1.4.0
2. 无数据库结构变更，安装后即可使用

### 手动覆盖

1. 下载 [misc-api1.4.0.zip](https://gitee.com/xunjinlu/misc-api/releases/download/v1.4.0/misc-api1.4.0.zip)
2. 覆盖站点文件（**保留** `config/database.php`、`config/install.lock`）
