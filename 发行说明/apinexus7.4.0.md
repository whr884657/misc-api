# ApiNexus 7.4.0 发行说明

- 公告 / 文章管理列表：桌面固定行高、标题单行省略、操作按钮靠右
- Markdown 卡片/按钮/时间轴强调色钉死黑白灰（后台与默认主题）；主题二仍显式青绿
- 规范：E89 / E90 / E91；列表取数统一禁止「近几天」
