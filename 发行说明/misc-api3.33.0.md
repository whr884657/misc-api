# misc-api 3.33.0

## 变更说明

- 接口支持「免费 / 收费」与单次扣费积分；收费调用须有效密钥且余额足够
- 用户积分余额、`orders` 订单流水（充值 / API 扣费 / 管理员加减款；AI 扣费预留）
- 码支付对接：后台支付配置、用户充值中心扫码、异步回调入账
- 管理员用户管理可调整积分；API 列表/审核展示扣费信息

## 升级说明

- **须执行数据库结构更新**（迁移 `3.33.0`）
- 升级后请到「交易财务 → 支付配置」填写码支付网关；异步通知由下单参数自动携带为 `paynotify.php`（v3.34.0；旧 `pay/notify.php` / `codepay.php` 路径已废弃）
- 建议清理浏览器缓存后验证充值与收费接口调用
