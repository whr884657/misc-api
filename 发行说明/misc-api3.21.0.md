# misc-api 3.21.0 发行说明

**日期：** 2026-07-17  
**类型：** 中版本（接口列表/审核/用户管理卡片 UI 重构 + 搜索分页）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.21.0/misc-api3.21.0.zip

## 变更

1. **卡片布局（管理端列表/审核、用户端 API 管理）**
   - 右上角标签横向排列（靠右往左）
   - 名称与编号上移，下方完整调用链接（绿色）
   - 元信息：状态 / 提交者 / 请求次数（用户端不含提交者）
   - 底部分隔线下按钮单行横排、略缩小
   - 「代理」「本地」标签均显示（用户端不强调类型标签）

2. **管理端接口列表功能**
   - 搜索：名称 / 地址 / 提交者
   - 分页：可选每页 10/20/50/100（手机默认 10，电脑默认 20）
   - 排序：接口 ID 倒序（大号在前）
   - 标题旁统计：仅总数，或总数 + 维护中 / 待审核明细

## 升级注意

- 无数据库结构变更。
- 刷新管理后台与用户中心页面即可（注意浏览器缓存 CSS/JS）。

## 库变更

否。
