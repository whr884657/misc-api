# misc-api 3.11.0 发行说明

**发布日期：** 2026-07-15  
**类型：** 中版本（接口本地/代理双模式、邮件通知开关、用户中心投稿修复）

## 下载

- ZIP：https://gitee.com/xunjinlu/misc-api/releases/download/v3.11.0/misc-api3.11.0.zip
- 标签：`v3.11.0`

## 变更摘要

1. **本地接口 / 代理外链**  
   - 本地：填写本站路径（如 `/api/img/index.php`）  
   - 代理：填写对方完整 URL；系统生成 `/proxy.php?s={短码}`，用户请求时 **纯 PHP 302** 跳转并附带查询参数（无需改 Nginx 伪静态）

2. **投稿权限**  
   - 管理员：可选本地 / 代理  
   - 已与管理员绑定的开发者：同上  
   - 未绑定开发者：仅能投代理外链（界面不显示类型切换）

3. **用户中心 API 管理**  
   - 修复「提交无反应」（弹窗须 `is-open`）  
   - 「提交接口」移至标题右上角；响应式大弹窗（手机抽屉 / 电脑居中）

4. **邮件通知开关**（系统设置 → 邮箱发信）  
   - 投稿通知管理员 / 审核通过通知用户 / 审核拒绝通知用户，可分别开关

## 升级说明

1. 更新至 v3.11.0  
2. **系统管理 → 系统升级**，执行结构更新（`3.11.0`）  
3. 在邮箱设置中按需调整通知开关  
4. 代理公开地址示例：`https://你的域名/proxy.php?s=短码`

## 库变更

| 项 | 说明 |
|----|------|
| `api.apitype` | 0 本地 / 1 代理 |
| `api.targeturl` | 代理上游完整地址 |
| `api.proxyslug` | 代理短码 |
| `config.mail_notify_submit/pass/fail` | 通知开关，默认开启 |
