# ApiNexus 4.6.0

**发布日期：** 2026-07-20  
**下载：** [apinexus4.6.0.zip](https://gitee.com/xunjinlu/apinexus/releases/download/v4.6.0/apinexus4.6.0.zip)

## 变更说明

- **全部接口页**：API 卡片与首页对齐（积分/KEY/Method 标签、分类色、端点容器配色）；计费数据不再误显示「免费」
- **页面标题**：全部接口 / 文章 / 贡献者 / 友链 / 赞助 / 关于等页标题顶距与文章页统一
- **在线测试**：按 Content-Type 安全渲染图片/音频/视频；禁止把二进制当文本塞进 DOM（防卡死）
- **详情页**：去掉 API ID；计费显示「N 积分 / 次」；接口链接复制按钮同行；参数示例作 placeholder；底部改为随机推荐接口卡；JSON 语法高亮
- **首页 Hero**：两个标签（默认「完全免费」「无需密钥」）可在主题设置中自定义
- **友链页**：空态去掉重复「申请友链」按钮
- **外链弹窗**：开合动画入库

## 升级说明

- 无数据库结构变更（`db_changes: false`）
- 建议清空前台接口 Redis 缓存后查看全部接口页标签
