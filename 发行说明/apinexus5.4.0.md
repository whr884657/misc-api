# ApiNexus 5.4.0 发行说明

**版本：** 5.4.0  
**日期：** 2026-07-21  
**类型：** 中版本（多页面 UI：标题 / 分页选择 / 加载动效）  
**数据库：** 无结构变更

## 变更

1. **首页标题**  
   双主题首页浏览器标题由「首页 - 站点名称」改为仅 **站点名称**。

2. **底栏「每页」选择**  
   - 订单 / 日志 / 积分 / API 列表等统一 `vs-pick`  
   - 底栏「每页」使用 **弹层（sheet）**，手机底部抽屉、电脑居中卡片  
   - 按钮高度与「上一页 / 下一页」同为 **32px**  
   - 管理后台与用户中心布局默认加载 `vs-pick.js`

3. **数据加载动效**  
   - 新增 `vs_render_loading()` / `VS.setLoading()` / `.vs-loading`  
   - 订单、积分、日志、主题配置等列表加载统一动效，不再只显示「加载中…」

## 升级

覆盖代码即可；**无需**数据库结构更新。
