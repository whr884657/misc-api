# ApiNexus 8.0.0 发行说明

**版本：** 8.0.0  
**日期：** 2026-07-24  
**数据库：** 无结构变更

## 变更摘要

- **管理员「接口列表」全面重构**（对齐参考设计）：电脑端白底数据表格，手机端独立信息卡
- 标题行保留搜索与「添加接口」；下方增加分类 / 状态 / 排序筛选（前端静态过滤，URL 不变）
- 列表徽章：接口类型、收费、KEY 要求、状态；调用次数千分位展示
- 操作随状态变化：正常→维护/禁用；维护中→恢复正常；已禁用→启用；保留编辑/删除
- 底部分页：每页 10/20/30/50 + `vs-pick` 弹层；底栏在面板外；控件高度统一 32px
- 编辑弹窗（基础/参数/文档）与 AJAX CRUD 逻辑保持不变
- 完善《列表底部分页规范》《开发规范与功能优化》及易错点 **E94**

**升级说明：** 覆盖更新即可；无需执行数据库结构更新。
