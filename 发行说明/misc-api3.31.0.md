# misc-api 3.31.0 发行说明

**日期：** 2026-07-18  
**类型：** 大版本（参数编辑体验 + 默认主题壳层修复 + 接口表单多项修正）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.31.0/misc-api3.31.0.zip

## 变更

1. **请求参数表格编辑**  
   管理后台 / 用户中心默认用表格填写（参数名、类型、必填、描述、示例）；可切换 JSON 粘贴并自动识别互转。返回参数仍为 JSON 文本。

2. **内置常见参数类型**  
   字符串、整数、布尔、对象、URL、邮箱、UUID、枚举等，JSON 中的别名类型会归一到内置项。

3. **默认主题用户中心 UI 修复（E25）**  
   根因：主题内过期复制的 `user.css` 未同步 `admin.css`。现统一加载公共 `admin.css`，主题 `user.css` 仅保留增量。

4. **管理员接口表单**  
   - 去掉「审核状态」控件（本页发布一律通过）  
   - 须绑定用户身份后发布；列表投稿者显示绑定用户名  
   - 「所属分类 / 是否需要密钥」下拉对齐  

5. **请求方式多标签着色**  
   GET、POST 同时存在时各自着色，不再合并成无样式的 `getpost` 类名。

## 升级注意

- 无数据库结构变更。  
- 管理员若未绑定用户账号，后台「添加接口」将提示先去账号设置绑定。  
- 强刷 CSS/JS（`admin.css`、`api-params-editor.js`、`api-list.js`、`user-api-manage.js`）。

## 库变更

否。
