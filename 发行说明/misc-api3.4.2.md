# misc-api 3.4.2 发行说明

**发布日期：** 2026-07-14  
**版本类型：** 小版本（`3.4.1` → `3.4.2`，Redis 饼图标注交互）  
**数据库变更：** 否

---

## 概述

Redis 监控统计图改为引出线标注饼图：默认在扇区旁显示数据；点击后强化对应扇区、引出线与标注，不再使用图下方提示文字。

---

## Redis 监控

| 状态 | 效果 |
|------|------|
| 默认 | 各扇区引出折线，线端显示「名称 数值 单位」与占比 |
| 点击 | 扇区外扩、引出线加粗、标注加粗加深；其余扇区变淡；圆心同步显示该板块 |
| 再点 | 取消高亮，恢复默认标注 |

- 已移除卡片下方 tip 提示条
- 缓存项状态图中心仍默认显示缓存占用
