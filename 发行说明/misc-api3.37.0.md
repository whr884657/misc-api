# misc-api 3.37.0

## 变更说明

- **分页同高（强制）**：底栏「每页」与上一页/下一页统一 32px，覆盖 `.vs-input` 默认 40px；列表工具栏按钮同步 32px
- **日志查询**：搜索工具栏移出标题行（修复手机标题挤成方块）；列表/卡片带「查看」；GET/POST 等方法色与状态码色；详情展示 IP 归属地（空则 —）、用户名、管理员明文密钥；首屏 PHP 预取减少空白等待
- **Redis**：环形图按缓存业务项展示「缓存了什么」；列表文案可读化；去掉「轻量」；日志查询 + 今日调用次数均走短 TTL 缓存
- **积分变动**：手机端标题与刷新同行；管理员后台显示完整密钥（用户中心仍脱敏）

## 升级说明

- 无数据库变更；覆盖后请强刷后台 CSS/JS（Ctrl+F5）
