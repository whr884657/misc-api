# misc-api 3.35.0

## 变更说明

- 支付配置：去掉「码支付网关」标题；面板补齐左右内边距
- 支付图标：支付宝 / 微信 / QQ 使用 `assets/img` 下 SVG；扫码弹窗中心图标同步
- 订单管理：标题居左、筛选与刷新居右；按钮同高 36px
- 列表分页：每页选择与上一页/下一页统一 32px（管理员订单/积分、用户积分等）
- 令牌：新密钥前缀改为小写 `sk-`
- 用户充值 / 积分：去掉半屏留白；英雄条与套餐色走主题 CSS 变量；主题二青绿差异化

## 升级说明

- 无数据库变更；覆盖升级即可
- 建议强刷后台与用户中心 CSS/JS
- 已有旧令牌仍为 `SK-` 前缀可继续使用；新建/重置后为 `sk-`
