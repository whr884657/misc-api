# misc-api 3.23.0 发行说明

**日期：** 2026-07-17  
**类型：** 中版本（接口列表真分列 + 手机满宽右靠 + 审核侧栏红点）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.23.0/misc-api3.23.0.zip

## 变更

1. **电脑端**  
   管理端列表/审核、用户端 API 管理改为**横向真分列**：图标 → 名称+#ID → 方法+链接 → 标签 → 状态 → 请求 → 提交 → 操作，垂直居中对齐，不再上下堆叠。

2. **手机端**  
   卡片满宽少留白；标签靠右上；状态/请求/提交与按钮靠右；虚线分隔拉满卡片宽度。

3. **分页**  
   「每页」选择与上一页/页码/下一页在同一行；「当前接口总数」仍在底部分页下方居中。

4. **按钮**  
   正常/维护/禁用常显，当前状态高亮变色；审核页通过/不通过同理。

5. **审核红点**  
   存在待审投稿时：侧栏「API 管理」折叠显示红点，展开后红点在「接口审核」。细则见本地《侧边栏红点提醒规范》。

## 升级注意

- 无数据库结构变更。  
- 请强刷后台与用户中心缓存。

## 库变更

否。
