# ApiNexus 4.3.0

**发布日期：** 2026-07-19

## 变更摘要

- **主题切换**：主题卡片居中；去掉画廊外层多余边框
- **默认主题首页**：打字机两行文案可在「主题设置」中自定义
  - 第1行：`首页第1行 · 主标题`（默认站点名称）
  - 第2行：前缀 / 高亮 / 后缀三段（默认「开发者的」「开放 API」「 接口平台」）
  - 下方长文案仍用「首页说明段落」

## 配置入口

后台 → 主题设置 → **主题设置** Tab（需先切换并保存为默认主题）→ 上述字段。

## 升级说明

无数据库结构变更。
