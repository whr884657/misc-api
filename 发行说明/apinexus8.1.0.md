# ApiNexus 8.1.0 发行说明

**版本：** 8.1.0  
**日期：** 2026-07-25  
**数据库：** 无结构变更

## 变更摘要

- **接口审核页全面重构**（对齐参考设计）：电脑端白底表格（名称 / 开发者 / 提交时间 / 状态 / 操作）；手机端独立信息卡
- **筛选与搜索**：待审核 / 已通过 / 未通过 Tab；标题行搜索框；前端静态过滤，**地址栏无查询串**
- **分页**：底栏在内容区、panel 外；每页 10/20/30/50 + `data-vs-pick="sheet"`；控件高度 32px
- **手机操作按钮靠右**：审核页与接口列表页卡片操作区统一 `justify-content: flex-end`
- **避坑**：搜索样式挂 `.vs-api-list-toolbar`（防放大镜飞出，E95）；主区 `overflow-x: clip` 防横向空白

**升级说明：** 覆盖更新即可；无需执行数据库结构更新。
