# misc-api 3.24.0 发行说明

**日期：** 2026-07-17  
**类型：** 中版本（接口详情、多选请求方式、自定义选择面板、外链代理、后台去 .php）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.24.0/misc-api3.24.0.zip

## 变更

1. **请求方式**  
   管理端 / 用户端支持同时勾选 GET 与 POST。

2. **密钥说明**  
   「完全不需要」与「可选」调用规则相同；表单增加备注说明差异主要在前台是否展示填写框。

3. **自定义选择面板（vs-pick）**  
   分类、状态、审核、密钥、每页条数等单选字段不再依赖浏览器原生下拉观感。

4. **接口审核按钮**  
   已通过仅「编辑 + 不通过」；未通过仅「编辑 + 通过」；待审核三者齐全。

5. **接口详情页**  
   双主题新增详情页；出站地址 `/api-detail/{id}`（需更新 Nginx 伪静态，见根目录说明）。

6. **外链媒体代理**  
   头像 / 图标外链经 `/media-proxy` 同源展示，降低跨域与防盗链影响。

7. **后台 URL**  
   侧栏、顶栏、登录跳转去掉 `.php`；登录 / 忘记密码去掉与 placeholder 重复的 label。

8. **默认主题 UI**  
   全部接口顶栏留白收紧；贡献者 / 赞助页标题改为左上统一风格。

## 升级注意

- 无数据库结构变更。  
- **务必**在站点伪静态中增加 `/api-detail/` 规则（见 `nginx伪静态配置.md`）。  
- 强刷后台与前台静态资源缓存。

## 库变更

否。
