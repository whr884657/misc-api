# misc-api 2.12.1 发行说明

**发布日期：** 2026-07-12  
**类型：** 小版本（主题二体验修复与命名统一）

---

## 变更摘要

### 主题二命名

- 对外名称由「青绿平台」统一改为 **「主题二」**（内部 ID 仍为 `slate`）
- 默认主题名称保持 **「默认主题」** 不变

### 导航 FAB 修复

- 修复保存「右下角圆形按钮 · 向上弹出」后刷新不显示的问题
- `themeSetting` 合并 schema 默认值；FAB 在选中该模式后**全端可见**
- 手机端 FAB 模式下隐藏顶栏三横线与主导航；电脑端保留顶栏导航 + FAB

### 顶栏交互

- Logo 与站点名称分离：**仅站点名称**点击跳转首页，避免顶栏左侧大面积误触

### 首页视觉

| 项目 | 说明 |
|------|------|
| 主标题 | 留空时显示「欢迎使用 {站点名称}」 |
| 副文案 | 打字机逐字输出（支持 `prefers-reduced-motion` 降级） |
| 统计区 | 三卡片网格布局，替代原胶囊横条 |
