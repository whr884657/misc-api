# misc-api 3.19.0 发行说明

**日期：** 2026-07-16  
**类型：** 中版本（接口管理 UI 大改 + `apilog.iploc`）

## 下载

https://gitee.com/xunjinlu/misc-api/releases/download/v3.19.0/misc-api3.19.0.zip

## 变更

1. **调用日志 `iploc`**  
   在客户端 IP 后增加「IP 中文归属地」字段（预留）。本版写入空字符串；后续可在系统设置开启 IP 解析后再自动填写。界面设置暂未做。

2. **本地统计注入（站长向）**  
   - 推荐写法：自当前目录向上查找 `core/bootstrap.php`，再 `ApiStats::hit()`  
   - 兼容 PHP 7.4～8.2，任意 `api/` 子目录深度  
   - 说明文档：`api/统计代码使用说明.md`  
   - 示例：`api/yiyan/v1.php`、`api/yiyan/aword.php`

3. **管理端接口列表**  
   紧凑卡片：接口 ID、类型（本地/代理）、状态、完整调用链接（代理显示本站 `/apis/短码` 绝对地址）、调用次数、操作按钮。

4. **管理端接口审核**  
   - 仅「待审核 / 已通过 / 未通过」，去掉「全部」，进入默认待审核  
   - 修复筛选后仍显示全部（`[hidden]` 被 grid 覆盖）  
   - 操作：编辑（跳转列表编辑）、通过、不通过  
   - 显示完整调用链接与投稿者

5. **用户中心 API 管理**  
   - 不显示本地/代理类型  
   - 展示 ID、状态/审核、请求方式、完整链接、调用次数  
   - 已通过审核的接口可切换正常 / 维护 / 禁用；保留编辑、删除

## 升级注意

- 后台「系统升级」执行**数据库结构更新**（应用 `3.19.0` 迁移，为 `apilog` 增加 `iploc`）。
- 本地接口请按 `api/统计代码使用说明.md` 粘贴统计代码（勿再手写目录层数）。

是（`{prefix}apilog` 增加 `iploc`）。
