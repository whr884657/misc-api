# ApiNexus 7.0.0 发行说明（补发）

**版本：** 7.0.0  
**说明：** 本包为 **补发跳板包**：在原 7.0.0 能力上，同步带上修复后的在线更新 / 数据库迁移逻辑（E83），避免 6.x→7.0.0 时仍使用旧 Updater 导致不跑库。

## 升级注意

1. 从 6.x 升到本包后应出现 `content` 表；完成提示须含「已同步数据库结构」或「结构已就绪」
2. 若曾回退版本号，刷新后台后会清理 `schema_migrations` 中高于代码版本的记录
3. 升完 7.0.0 后请继续升到 **7.1.0**（封面布局等）
