# ApiNexus 5.0.0

**发布日期：** 2026-07-20  
**下载：** [apinexus5.0.0.zip](https://gitee.com/xunjinlu/apinexus/releases/download/v5.0.0/apinexus5.0.0.zip)

## 变更摘要

- **友情链接禁用：** 审核通过后可禁用/重新启用；禁用不改变审核状态，前台列表与页脚不再展示。
- **合作伙伴共用表：** 与友链共用 `link` 表，字段 `kind`（0 友链 / 1 合作伙伴）、`enabled`（0 禁用 / 1 启用）；合作伙伴无审核，管理员直接维护。
- **后台：** 内容运营 → 合作伙伴（编辑 / 启禁 + 桌面弹窗 / 手机抽屉）。
- **默认主题：** 首页合作伙伴区改为 `FrontendPartner::listForTheme()`。
- **手机端：** 友链后台行卡片独立边框、操作按钮靠右；前台友链描述限两行，申请按钮靠右。

## 升级注意

本版含数据库结构变更（`db_changes: true`）。升级后请在后台「系统升级」中执行数据库结构更新。
