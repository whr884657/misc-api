# misc-api 3.3.0 发行说明

**发布日期：** 2026-07-14  
**版本类型：** 小版本（多模块 UI + Redis 优化）  
**数据库变更：** 否

---

## 概述

本版本统一全站 Toast 与后台按钮视觉规范，修复默认主题注册身份滑块 CSS、主题二认证页布局、后台主题切换选中态，并增强 Redis 管理页数据可视化与键空间自动维护。

---

## Toast 与按钮

| 变更 | 说明 |
|------|------|
| Toast | 长条形、四角圆角；手机端左右间距；全站 `VsToast` 统一 |
| 后台按钮 | `.vs-btn` 改为圆角矩形，禁止胶囊形主按钮 |
| 开发规范 | 新增 `开发规范/按钮样式规范.md`；更新 `界面提示规范.md` §8 |

前台特殊主题（默认 / slate）认证主按钮仍可保持胶囊形（规范例外）。

---

## 主题修复

- **主题 1**：`vs-role-segment` 样式写入主题包 `assets/auth.css`（此前误写在未加载的 `auth-login.css`）
- **主题 2**：认证页去掉用户名/邮箱等上方 label，placeholder 即提示
- **后台主题切换**：点击未保存主题后 `.is-selected` 黑框不随鼠标移开消失

---

## Redis

- 管理页新增三块 CSS 饼图：命中分布、键类型、缓存项状态
- 文案不再绑定固定站点名称
- `RedisCache::maintainKeyspace()`：清理过期限流键、限制键数量、防止统计无限增长
