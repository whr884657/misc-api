# ApiNexus 7.3.0 发行说明

**版本：** 7.3.0  
**日期：** 2026-07-23  
**数据库：** 无结构变更

## 变更摘要

- 默认主题全站粒子统一为灰点（`shell.js` 读 `--particle-color`），去掉关于/文章/贡献者/文档等页重复绿色粒子脚本
- 清除页面绿色 `:root` / 网格覆盖，对齐 `theme-tokens` 黑白灰色板
- 贡献者页去掉终端乱码引导条，缩短介绍文案
- Markdown 引用改为中文弯引号 `“ ”` 装饰样式，强调色跟随主题（禁止绿色竖线）
