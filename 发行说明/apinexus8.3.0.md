# ApiNexus 8.3.0 发行说明

**版本：** 8.3.0  
**日期：** 2026-07-25  
**数据库：** 无结构变更

## 变更摘要

- **根因修复（E98）**：废弃旧 `.vs-api-cat-row { display:grid }`，不再压垮新版真实 `<table>` / `.cat-card`
- **电脑端**：白底表格卡片，分类名称（图标+文字）正常显示，操作按钮横排靠右
- **手机端**：独立信息卡对齐参考设计（左名称、右「N 个」、描述左对齐、操作靠右）
- **分页**：底栏「每页」+ 翻页键同行，控件高度 32px

**升级说明：** 覆盖更新即可；无需执行数据库结构更新。建议强刷后台 CSS/JS 缓存。
