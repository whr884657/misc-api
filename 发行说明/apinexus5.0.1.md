# ApiNexus 5.0.1

**发布日期：** 2026-07-20  
**下载：** [apinexus5.0.1.zip](https://gitee.com/xunjinlu/apinexus/releases/download/v5.0.1/apinexus5.0.1.zip)

## 变更摘要

- **合作伙伴删除：** 后台支持删除合作伙伴记录。
- **静态更新（强制）：** 友链 / 合作伙伴的添加、编辑、通过/拒绝、启禁、删除一律 AJAX + 局部 DOM，**禁止** `location.reload()`。
- **电脑端 UI：** 列表改为独立卡片（描边 + 圆角），与手机端视觉语言一致；宽屏操作按钮靠右、状态居中。

无数据库结构变更。
