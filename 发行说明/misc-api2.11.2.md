# misc-api 2.11.2 发行说明

**发布日期：** 2026-07-12  
**类型：** 小版本（青绿主题认证页 UI 还原）

---

## 变更摘要

### 还原 2.10.2 前的 slate 认证页视觉

| 项目 | 说明 |
|------|------|
| 布局 | 恢复 **无卡片居中** 单栏布局（非 2.11.x 双栏重构） |
| 标题 | 右对齐 + 轻微 pulse 动效 |
| 背景 | 浅绿渐变 + 缓慢浮动光球 |
| 输入框 | 底部边框线样式（非圆角卡片输入框） |
| 安全 | **保留** mail_ticket、120s 倒计时、parseWaitSeconds 等 2.10.2 能力 |

### 小幅优化（非重构）

- 桌面端表单 max-width 由 400px 调整为 420px
- 「记住账号 / 忘记密码」行增加 flex-wrap，避免链接被截断

涉及：登录、注册、忘记密码、OAuth 绑定页。
