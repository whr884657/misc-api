# ApiNexus 7.1.0 发行说明

**版本：** 7.1.0  
**日期：** 2026-07-23（含补发）  
**类型：** 中版本 + 同号补发（更新机制 / Markdown 视频 / 公告弹窗）  
**数据库：** 有结构变更（`content.coverlayout`）

## 变更

- **Markdown 主题色**：引用/卡片/按钮跟随主题变量（E80）
- **公告 / 文章后台**：卡片对齐 API 列表；公告无摘要、无预览；文章封面左/右/背景
- **分类删除**：有接口时可转移后再删
- **【补发】在线更新（E81）**：`db_changes` 必须 ensure + 结构校验，禁止误报「无数据库结构变更」；升级页新增「执行数据库结构更新」
- **【补发】视频短码（E82）**：`@[video]` 插槽渲染，不再被 SafeMode 转义成乱码
- **【补发】公告弹窗**：「不再提示」（白底）+「我知道了」；仅前者写入本地存储

## 升级注意

1. 后台执行 **数据库结构更新**；若版本号已是 7.1.0 但缺 `coverlayout`，请点系统升级页「执行数据库结构更新」
2. 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v7.1.0/apinexus7.1.0.zip
