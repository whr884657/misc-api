# misc-api 3.13.0 发行说明

**发布日期：** 2026-07-15  
**类型：** 中版本（代理投稿表单修复、slate 布局、在线更新清理废弃文件）

## 下载

- ZIP：https://gitee.com/xunjinlu/misc-api/releases/download/v3.13.0/misc-api3.13.0.zip
- 标签：`v3.13.0`

## 变更摘要

1. **代理外链保存校验**  
   选择代理类型时不再要求「本地路径」，改为校验上游完整地址与短码。

2. **接口短码必填**  
   后台与用户中心均须填写 3～64 位字母或数字短码（不再自动生成）。

3. **投稿/添加弹窗字段顺序**  
   名称优先；图标选择与图标链接移至表单末尾。

4. **主题 slate 用户中心**  
   修正标题区与「提交接口」按钮布局，避免与下方内容面板重叠导致点击无效。

5. **在线更新清理废弃文件**  
   文件覆盖完成后，读取更新包内 `install/obsolete-files.json`，删除站点上残留的废弃文件（如历史 `proxy.php`）。受保护路径（`config/database.php`、`config/install.lock`、`data/`）绝不会被删除。

## 升级说明

1. 更新至 v3.13.0（在线更新或覆盖 ZIP）  
2. 本次**无**数据库结构变更  
3. 若此前残留 `proxy.php`，在线更新部署步骤会尝试自动删除；手工覆盖者可手动删除根目录 `proxy.php`  
4. 以后新增废弃文件时，将其相对路径写入 `install/obsolete-files.json` 的 `files` 数组

## 库变更

无。
