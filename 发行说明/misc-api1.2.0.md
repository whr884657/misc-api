# misc-api 1.2.0 发行说明

**发布日期：** 2026-07-11  
**版本类型：** 次大版本（功能较多：修复 + UI + 新功能 + 移除旧功能）

## 更新摘要

- 修复系统升级页更新记录始终为空的问题
- 全站统一界面提示样式（`vs-notice`）
- 新增注册邮箱后缀白名单
- 移除多域名/子域名绑定（API 系统仅需单域名）

## 详细变更

### 修复

- `UpdateLog` 现正确解析 `update-log.json` 中以版本号为 key 的对象结构，升级页可正常展示全部历史版本

### 界面

- 新增 `vs_render_notice()` 与 `.vs-notice` 组件
- 系统升级、控制台、账号设置、用户注册等页面提醒样式优化
- 移除升级页「优先从云端读取」等多余小字
- 根目录新增 [界面提示规范.md](../界面提示规范.md)

### 功能

- **注册邮箱后缀限制**：后台「系统设置 → 用户注册」配置允许的后缀（JSON 存于 `register_policy`），留空则不限制
- **移除子域名**：删除绑定子域名设置页、相关配置项与 `domain` 表（升级时由 1.2.0 迁移执行）

## 升级说明

1. 备份数据库与网站文件
2. 后台「系统升级」检测并安装 v1.2.0
3. 本版本含数据库变更（`install/migrations/1.2.0.sql`），会自动写入 `register_policy` 并清理域名相关数据

## 版本号规则

- **大版本（x.y.0）**：较多功能变更、结构调整
- **小版本（x.y.z）**：少量优化或修复
