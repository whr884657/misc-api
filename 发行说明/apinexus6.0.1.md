# ApiNexus 6.0.1 发行说明

**版本：** 6.0.1  
**日期：** 2026-07-22  
**类型：** 补丁（归属修复 + 贡献者/主页 UI）  
**数据库：** 有数据回填（历史 `userid=0` → 唯一绑定用户；无新字段）

## 变更

- **归属修复：** 管理员绑定某用户后，后台发布与用户中心发布视为同一作者；贡献者列表、个人主页、用户中心接口管理均计入（含旧版 `userid=0` 历史数据）
- **贡献者页：** 「衷心感谢」软横幅改为与默认主题一致的终端风格引导条
- **个人主页：** 接口列表改为 CSS Grid `auto-fill` 多列（双主题），不再写死两列

## 升级注意

1. 后台执行 **数据库结构更新**（会按版本跑 `6.0.1` 回填逻辑）
2. 若全站有多个不同绑定身份，历史 `userid=0` 不会自动挂载（避免误挂）；可在后台编辑保存该接口以补挂当前绑定用户
3. 下载：https://gitee.com/xunjinlu/apinexus/releases/download/v6.0.1/apinexus6.0.1.zip
