# ApiNexus 8.2.0 发行说明

**版本：** 8.2.0  
**日期：** 2026-07-25  
**数据库：** 无结构变更

## 变更摘要

- **接口分类页全面重构**（对齐参考设计）：电脑端白底表格（名称 / 描述 / 接口数量 / 状态 / 操作）；手机端独立信息卡
- **搜索与分页**：标题行搜索框；前端静态过滤，**地址栏无查询串**；底栏每页 10/20/30/50 + `data-vs-pick="sheet"`
- **手机操作按钮靠右**：与列表/审核页一致（`justify-content: flex-end`）
- **分类图标库**：删除全部英文文件名 SVG，仅保留数字名（如 `1.svg`…`209.svg`）；扫描与入库校验同步收紧
- **保留产品能力**：启用/禁用、删除时内联转移接口（禁止弹窗套 vs-pick）
- **避坑**：搜索样式挂 `.vs-api-list-toolbar`（防 E95）；主区 `overflow-x: clip`；见 E97

**升级说明：** 覆盖更新即可；无需执行数据库结构更新。若库中分类仍指向旧英文图标路径，将自动回退为数字默认图标。
